## **[ Формулировка задания ]**

Сверстать динамический список из разных по содержанию элементов, около 50 штук

- Элемент списка содержит иконку, заголовок
- Индикация, что элемент выделен (рамка или цвет)
- Выделение управляется стрелками клавиатуры
- При нажатии на элемент мышкой или на определенную клавишу клавиатуры, элемент плавно разворачивается или сворачивается, отображает второй текст
- Список должен иметь скролл, двигаться касанием и, при управлении с клавиатуры, двигаться к выделенному элементу
- Дизайн элементов должен содержать градиент и скругления углов
- Данные для отображения можно взять произвольные
- Использовать можно MonoBehaviour и классы, без сторонних библиотек (совсем: ни фреймворков, ни DOTween)
- Оцениваться будет аккуратность оформления кода, сцены и префабов
